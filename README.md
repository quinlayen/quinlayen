Data Engineer and Software Developer
------------------------------------

I'm a 15 year commercial real estate veteran in both the Hawaii and Tokyo markets. I discovered software development later in life and I never looked back. There is no other feeling than struggling with a problem and then it..clicks. -Thank you rubber ducky. 

Originally I was trained in full-stack web development with JS and React on the front and Node and Postgres in the back. After that I worked for 5 years in the game development industry using Unity and Lumberyard. Great fun, but something about data from my original years stuck with me. Now I'm a Data Engineer!
